This script is intended for red team research and defensive learning. Its purpose is to help understand how ransomware techniques operate at a technical level and, more importantly, how such behavior can be detected, mitigated, and prevented.
The code is maintained in a private repository by design and is not intended for public distribution or general use.
All usage is strictly limited to educational purposes and controlled lab environments on systems that are owned by, or explicitly authorized for testing by, the user
